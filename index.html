  • Unsupervised Latent Dirichlet Allocation Assisted Github Readme File Content Classification

    • This project centered around a thesis on machine learning and natural language processing. Its objective was to enhance the quality of GitHub readme files through the utilization of machine learning algorithms. The initial step involved gathering a dataset of GitHub readme files by means of web scraping from the GitHub platform. These files were subsequently annotated manually, adhering to the official recommendations provided by GitHub. Moreover, to automatically label various sections within the readme files, a series of preprocessing steps were implemented. To extract meaningful topics from the readme files and facilitate the automatic labeling process, the latent Dirichlet allocation algorithm, a topic modeling technique, was applied in various configurations. .

    Available at: (Link)

The project involved obtaining a dataset from District Headquarter Hospital Jhelum, which consisted of symptom records from individuals diagnosed with dengue fever. Following a series of preprocessing steps to prepare the data, we employed machine learning algorithms to perform the task of diagnosing and detecting dengue fever in patients.
